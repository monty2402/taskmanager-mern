📌 Task Manager App (Frontend + DevOps Practice Project)

A modern Task Manager frontend application built using React with glassmorphism UI, animations, and Dockerized setup.
This project also includes DevSecOps practices like Trivy security scanning and multi-stage Docker builds.

🖥️ Tech Stack
React.js
JavaScript (ES6+)
CSS3 (Glassmorphism UI)
Framer Motion (animations)
Docker (containerization)
Trivy (security scanning tool)
